What is HTML?

Also known as Hyper Text Mark Up Language, HTML is used for defining and organizing the structure of the webpage, do not confuse it with the design aspect, as that is where CSS comes into play. You will learn how to setup a basic HTML document and printing some text to the screen.

DOCTYPE

The HTML document type declaration, also known as DOCTYPE, is the first line of code required in every HTML or XHTML document. The DOCTYPE declaration is an instruction to the web browser about what version of HTML the page is written in. This ensures that the web page is parsed the same way by different web browsers.
Reference the type of web Documentto your browser by typing the following:

<!DOCTYPE html>
HTML TAG

An HTML tag is a piece of markup language used to indicate the beginning and end of an HTML element in an HTML document. When creating tags in html you must note that there are opening and closing tags. Opening Tags specifies where the piece of code begins whereas the closing tag specifies where the code ends.

Create an opening HTML tag as shown: <html> Create a closing HTML tag as shown: </html> Your code should like like this: <!DOCTYPE html>
<html>

</html>
HEAD TAG

The head tag is a tag that contains metadata. MetaData is data that is used to describe and give information about the website. A few examples are, The website title, relationship references between different files, etc... In this course we wont be getting into detail about the head, however you will give your website a title

Setup your head tag in the middle of your html tags: <!DOCTYPE html>
<html>
 <head>
 </head>
</html>
TITLE TAG

If you look to the top of this/ any webpage you would notice in the tab there is a name for the website and an icon that describes the name of the website. This tag is pretty self explanitory, it gives your website it's title which can be found above in tab section as shown below.



Give your website a title with the <title> </title> tag

<!DOCTYPE html>
<html>
 <head>
  <title>Title Name</title>
 </head>
</html>
BODY TAG

This is the area in the document where all of the magic happens. The body tag is used for the main content - What the user will be visually seeing, this is where the text of website is made, images, videos, audio, templates etc...

Lets create your body tags followed by a paragraph tag to output some text to the screen
<!DOCTYPE html>
<html>
 <head>
  <title>Title Name</title>
  <body>
   <p>

This is a paragraph

</p>
  </body>
 </head>
</html>
